# Java基础

## final关键字的作用
1. 修饰类：表示该类不能被继承
2. 修饰方法：表示该方法不能被重写
3. 修饰变量：表示该变量的值不能被修改，必须在声明时初始化

## 什么是虚拟线程？它和平台线程有什么区别？
虚拟线程是Java 19引入的一种轻量级线程实现，属于Project Loom的一部分。虚拟线程与平台线程（操作系统线程）相比，有以下区别：
1. 轻量级：虚拟线程的创建和销毁开销较小，可以创建大量的虚拟线程而不会消耗过多的系统资源
2. 非阻塞：虚拟线程在等待I/O操作时不会阻塞底层的操作系统线程，从而提高了并发性能
3. 调度：虚拟线程由JVM调度，而不是操作系统调度，可以更好地利用CPU资源
4. 兼容性：虚拟线程可以与现有的Java代码和库无缝集成

## ==和equals的区别
1. ==比较的是两个对象的引用地址是否相同
2. equals比较的是两个对象的内容是否相同，默认实现也是比较引用地址
3. String类重写了equals方法，比较的是字符串的内容是否相同
